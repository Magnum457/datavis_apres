## Apresentação de DataVis
O seguinte repositório tem como propósito expor os dados apresentados na cadeira de Visualização de Dados.


**Instalação**

Instalando com npm
--------------------
```
npm install
```
Instalando com yarn
--------------------
```
yarn install
```



**Executando o projeto**

Executando com npm
--------------------
```
npm start
```

Executando com yarn
--------------------
```
yarn start
